# Drawback Chess AI

Local development repository for the Drawback-Chess-Engine project.

## Upstream Repository

This repository is wired to: https://github.com/titaniummachine1/Drawback-Chess-Engine

## Project Description

Engine made in MCTS using probability of moves in future being legal from machine learning agent learning how to read drawbacks.

## Development Setup

This is a local development environment for contributing to or working with the Drawback Chess Engine.

### Repository Structure

- `/src` - Source code for the chess engine
- `/ml` - Machine learning components
- `/tests` - Test suites
- `/docs` - Documentation

## Getting Started

1. Clone this repository
2. Install dependencies (see requirements.txt or package.json)
3. Run the main engine script
